Planering: Slutprojekt

Inledning

Beskrivning

Min plan är ett göra en digital formelbok för bland annat Fysik.

Kund

Fysikelver som lätt vill kunna nå den formeln som behövs för en viss uppgift (Viggo).

Betygsmål

Jag satsar på ett A för detta projekt.

Ingående beskrivning

Beskrivningen

Tanken är att man ska kunna markera vilka storheter man har och vilken man söker. Hemsidan kommer då ge formeln som kan ge svaret.

Lista på allt som skall göras

  • Skriva HTML-kod för att visa upp formler.
  • Skriva HTML-kod för att ta input för sökt/känd storhet från användaren.
  • Skriva CSS-kod för att få hemsidan att se bra ut.
  • Föra över relevanta formler till MathML-formatet.
  • Javscript som tar input och söker igenom en databas (objekt) av formler för att hitta en som har de sökta/kända storheterna och som dessutom inte har någon extra.
  • Om ingen formel matchar detta söker den igenom databasen igen för att hitta om en
  • Javascript som tar input och söker igenom en typ av databas för att hitta formeln eller formlerna som passar med de sökta och kända storheterna. Eventuellt även ställa om formlerna för att ge en färdig formel.

Annat

Hemsidan kräver ett skript som jämför inputen från hemsidan och jämför den med olika formler för att hitta den korrekta och kombinerar dem med varandra om det behövs. Eventuellt kombinerar skriptet olika formler vid behov. HTML-koden kommer vara relativt simpel då det enda som krävs från den är några fält för formler samt ett input-område. Slutligen kommer CSS-koden också vara simpel då jag inte planerar för att sidan ska vara relativt simpel inom den visuella aspekten.
